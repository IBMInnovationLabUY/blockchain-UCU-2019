Miembros

Los clientes pueden registrarse como miembros de este programa creando un número de cuenta para identificación en la red y tarjeta de acceso para iniciar sesión. Una vez registrados, pueden iniciar sesión, realizar transacciones para ganar puntos, usar puntos, y ver todas sus transacciones.

Partners

Para las empresas que forman parte del programa, pueden registrarse como partner en la red creando una identificación y una tarjeta de acceso para iniciar sesión. Pueden iniciar sesión para ver todas las transacciones realizadas con ellos y el panel de control para ver el total de puntos asignados y por Miembros.
